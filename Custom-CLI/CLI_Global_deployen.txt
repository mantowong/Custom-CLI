Problem: Eingabe von 'node index.js input', um mit CLI zu arbeiten
L”sung: In package.json unter  Property "bin" eine Property erstellen mit "ClI": "index.js"
	Dann 'npm i -g', um PATH variable anzulegen, sodass man im Terminal 'CLI Input' schreiben kann
